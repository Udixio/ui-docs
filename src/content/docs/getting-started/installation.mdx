Udixio UI is a set of open-source React user interface components
infused with the principles of [Material 3;](https://m3.material.io/)
these components will equip you with the tools to create intuitive user interfaces.

## Download
Run one of the following commands to add Udixio UI to your project:
<CodePreview code={`npm i @udixio/ui`}/>

## Dependencies
Udixio UI believes in using robust and proven tools to deliver a seamless and intuitive user interface experience. Each tool we incorporate has been selected with careful consideration for its features and benefits.

### Styles
We lean on the robust Tailwind CSS framework for crafting the stylistic aspects of your application.
- **TailwindCSS** –
To learn more and get it set up, follow the [official Tailwind documentation](https://tailwindcss.com/docs/installation)
```javascript
	content: [
		'./node_modules/@udixio/ui/**/*.js'
	]
```
- **Tailwind-material** –
For a seamless incorporation of Material design into your development, we suggest you utilize `@udixio/tailwind-material`. This resource is conveniently packaged for easy integration. Here's a straightforward way to include it in your toolkit:
``` bash
npm i @udixio/tailwind-material
```
Once installed, you can use it to easily add Material colors and fonts to your Tailwind configuration:
``` javascript
const materialTailwind = createMaterialTheme(
  {
    primary: '#6750A4', // Place your primary color here
    secondary: '#958DA5', // Place your secondary color
    tertiary: '#B58392', // Place your tertiary color here
  },
  'class' // your dark mode
);

const config: Config = {
  ...
  theme: {
    colors: {
      ...materialTailwind.colors, // Register Material colors
    },
    fontFamily: {
      ...materialTailwind.fontFamily, // Register Material font families
    },
  },
  plugins: [...materialTailwind.plugins], // Enable color states and dark mode
};
```
